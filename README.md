# 📚 Java Mentorship Journey  

> Repositório criado para organizar a jornada de aprendizado em Java, desenvolvimento de software e preparação para o mercado de tecnologia.  
>  
> O foco da mentoria é construir conhecimento sólido, entendendo não apenas o COMO, mas principalmente o PORQUÊ das decisões técnicas.

---

## 🎯 Objetivo da Mentoria  

Desenvolver uma base sólida em programação, evoluindo até a capacidade de construir aplicações reais em Java e se preparar para oportunidades no mercado de tecnologia.

---

## 🧠 Como funciona a jornada  

A mentoria é estruturada em três pilares:  

- 🧩 Fundamentos e Base Técnica  
- 🛠️ Aplicação Prática e Projetos  
- 💼 Preparação para o Mercado

## 📂 Estrutura do Repositório  

### ⚙️ Setup  
Configuração do ambiente de desenvolvimento e ferramentas necessárias para iniciar os estudos.

### 🗺️ Roadmap  
Plano de evolução com etapas claras da jornada de aprendizado e objetivos de progresso.

### 🧱 Fundamentos  
Conceitos essenciais da programação e funcionamento básico do desenvolvimento de software.

### 🧩 Lógica e Algoritmos  
Exercícios e conteúdos focados em desenvolver raciocínio lógico e pensamento computacional.

### 🧬 POO  
Fundamentos da orientação a objetos aplicados ao desenvolvimento em Java.

### 🗄️ Estruturas de Dados  
Principais estruturas utilizadas no desenvolvimento e quando utilizar cada uma.

### 🧼 Clean Code  
Boas práticas para escrita de código legível, organizado e sustentável.

### 🧪 Projeto Final
Acompanhamento do projeto acadêmico com explicação das decisões técnicas e evolução da implementação.

### 🚀 Projetos Extras  
Projetos práticos para consolidar conhecimento e construir portfólio.

### 💼 Entrevistas  
Preparação técnica e comportamental para processos seletivos.

### 📄 Currículo  
Construção e evolução do currículo e posicionamento profissional.

### 🧠 Soft Skills  
Desenvolvimento de habilidades comportamentais e mentalidade de aprendizado.

### 📓 Diário de Aprendizado  
Registro contínuo da evolução, aprendizados e desafios ao longo da jornada.

---

## 📈 Como medir evolução  

O progresso será acompanhado através de exercícios, projetos, revisões de código e evolução do entendimento conceitual.

---

## 🤝 Regras da Mentoria  

- Priorizar entendimento ao invés de decorar  
- Praticar de forma consistente  
- Perguntar sempre que houver dúvidas  
- Manter registro da evolução  
- Evoluir passo a passo  

---

## 🚀 Resultado esperado  

Ao final da jornada, o aluno será capaz de:  

- Entender os fundamentos da programação  
- Desenvolver aplicações em Java com boas práticas  
- Participar de projetos reais  
- Construir portfólio técnico  
- Se preparar para entrevistas  
- Continuar aprendendo de forma autônoma  

---

⭐ Este repositório será atualizado continuamente conforme a evolução da jornada.
