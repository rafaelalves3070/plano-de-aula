# 🧬 Programação Orientada a Objetos  
Fundamentos da orientação a objetos aplicados ao desenvolvimento em Java.

## 🎯 Objetivo  
Entender como modelar soluções utilizando conceitos de POO.

## 📚 O que será abordado  
- Classes e objetos
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Composição

## ✅ Resultado esperado  
Capacidade de modelar sistemas simples usando orientação a objetos.
