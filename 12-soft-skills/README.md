# 🧠 Soft Skills  
Desenvolvimento de habilidades comportamentais essenciais.

## 🎯 Objetivo  
Desenvolver mentalidade e comportamento alinhados ao crescimento profissional.

## 📚 O que será abordado  
- Mentalidade de aprendizado
- Comunicação
- Resolução de problemas
- Organização de estudos
- Como aprender melhor

## ✅ Resultado esperado  
Maior autonomia e maturidade profissional.
