# 📅 Roadmap de Progressão – Mentoria Java | Rafael
Este roadmap foi estruturado para levar o aluno do nível fundamental até a construção de um sistema completo, <br />
aplicando boas práticas de engenharia de software utilizadas no mercado.

A progressão segue a lógica:
> Fundamentos → Orientação a Objetos → Qualidade de Código → Arquitetura → Sistema Distribuído → Mercado

---
## 🧠 Objetivo Final

Ao final da mentoria, o aluno deverá ser capaz de:
- Desenvolver uma API estruturada
- Modelar banco de dados relacional
- Aplicar princípios SOLID
- Escrever testes unitários
- Trabalhar com mensageria
- Containerizar aplicações
- Explicar decisões arquiteturais em entrevistas

---
## 📚 Plano de Progressão (12 Semanas – 2 Aulas por Semana)

| Semana | Aula 1 | Aula 2 | Exercícios | Objetivo Cognitivo |
|--------|--------|--------|------------|-------------------|
| 1 | Setup Ambiente + Git | Lógica de Programação (algoritmos e fluxo) | Exercícios de lógica básica | Aprender a pensar como programador |
| 2 | Variáveis, Tipos e Memória | Condicionais e Loops | Lista progressiva | Construir raciocínio lógico |
| 3 | Métodos, Escopo e Execução | Debug e resolução prática | Mini desafios | Entender como o código executa |
| 4 | Estruturas de Dados (Array, List, Map) | Complexidade básica (Big-O intuitivo) | Exercícios comparativos | Escolher estrutura correta |
| 5 | POO – Classes e Objetos | Encapsulamento e Modelagem | Modelagem de domínio simples | Pensar orientado a objeto |
| 6 | Composição vs Herança | Introdução ao SOLID (SRP, OCP) | Refatoração prática | Design consciente |
| 7 | Clean Code + Code Smells | Design Patterns (Strategy, Factory) | Refatoração guiada | Escrever código profissional |
| 8 | Testes Unitários (JUnit + Mockito) | Introdução a TDD | Criar testes para projeto base | Pensar em qualidade |
| 9 | Introdução a API REST | Estrutura em camadas (Controller, Service, Repository) | CRUD simples | Entender arquitetura web |
| 10 | Banco de Dados (Modelagem + JPA) | Validações + Exception Handling | Integração com banco | Persistência estruturada |
| 11 | Conceito de Mensageria | Docker + Containerização | Containerizar aplicação | Pensar em sistema distribuído |
| 12 | Fundamentos de System Design | Simulação de entrevista técnica | Defesa do projeto | Pensar como engenheiro |
