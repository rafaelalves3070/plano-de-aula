# 🧼 Clean Code  
Boas práticas para escrita de código legível e sustentável.

## 🎯 Objetivo  
Aprender a escrever código claro, organizado e fácil de manter.

## 📚 O que será abordado  
- Naming
- Funções pequenas
- DRY e KISS
- Code smells
- Introdução ao SOLID

## ✅ Resultado esperado  
Código mais legível e com melhor qualidade.
