# ⚙️ Setup  
Esta seção contém o guia para configuração do ambiente de desenvolvimento necessário para acompanhar a mentoria.

## 🎯 Objetivo  
Garantir que o ambiente esteja configurado corretamente para desenvolver projetos em Java sem bloqueios técnicos.

## 📚 Obrigatório Back
- Instalação do Java LTS
- Configuração de IDE: Intellij Community
- Git e GitHub
- Ferramentas necessárias (Maven, Git, Draw.io)
- Como rodar projetos localmente

## 📚 Opcional Front (projeto futuro)  
- Configuração de IDE: VsCode
- Instalação Node + npm
- Angular Cli

## ✅ Resultado esperado  
Ambiente pronto para desenvolver, executar e versionar código.
